Formulários

O HTML possui a tag form que possibilita definir formulários para submissão de dados informados pelo usuário por meio de campos de input, dentro do formulário. A tag <form> possui além dos atributos globais e também atributo acrtion="url", utilizado para definir o destino dos dados submetidos. Exemplo: 

            <form action="url">
                <input type="text" name="variavelA"
                ...
            </form>

Table

A tag table é utilizada para a definição de tabelas no HTML, internamente em uma tabela podemos criar linhas "rows" utilizando a tag <tr> e células dentro das linhas utilizando a tag <td> Exemplo:

        <table> 
            <tr> 
                <td> Célula 1x1 </td> <td> Célula 1x2 </td> 
            </tr> 
            <tr> 
                <td> Célula 2x1 </td> <td> Célula 2x2 </td> 
            </tr> 
        </table>
O código anterior define a tabela abaixo
Célula 1x1 Célula 1x2
Célula 2x1 Célula 2x2

Além disso podemos manipular o número de espaçoes utilizando em linhas e colunas por uma célula utilizando rowspan e colspan.
