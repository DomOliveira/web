O que é HTML?

HTML é uma liguagem de marcação que gera arquivos de texto para documentação e troca de informação.


!DOCTYPE é o comando que vai identiicar o tipo de arquivo que o navegar tem que rendenrizar


(html) é o que traduz o seu código para a leitura do navegador


(head) é o cabeçalho do html ele vai comportar tag to tipo style, incorporação de arvivo css etc


title intitula o cabeçalho, normalmente o que nós vemos no navegador como "guia"


Body comporta o "texto" do seu HTML é o corpo dele, onde você vai direcionar todo o modelo do seu site

  • HTML
  • CSS
  • JavaScript

A internet é uma interconexão de redes de computadores que possibilita a oferta de serviços variados entre eles: email, mirc, dns, ssh, telnet, web etc. A WEB por outro lado é um serviço que está disponibilizado na internet onde se utiliza documentos linkados escritos sob a liguagem HTML e o protocolo HTTP atendendo as requisições dos navedagores em um sistema cliente-servidor.
