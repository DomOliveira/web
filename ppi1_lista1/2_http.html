O que é o protocolo HTTP?

HyperText Transfer Protocol é o conjunto de regras que definem a comunicação na WEB onde define como o navegador pede e como o servidor entrega as páginas da Web



Como é usado na Web?

O cliente navegador faz uma requisição. O servidor recebe esse pedido e envia uma resposta HTTP, que pode ser:

  • o códido HTML da página
  • uma imagem
  • um erro (como "404 Not Found")

Resumo

O request contém o método, cabeçalhos e dados enviados pelo cliente. O response traz o código de status, cabeçalhos e o conteúdo retornado pelo servidor.

Versão do protocolo Indica a versão do HTTP usada
Host Nome do servidor (domínio) que está sendo acessado.
