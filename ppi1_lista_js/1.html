O que é JavaScript?

Motivação para a criação do JavaScript: JavaScript foi criado no final da década de 1990 para tornar páginas web estáticas interativas. Enquanto HTML estrutura conteúdo e CSS cuida do visual, JavaScript foi pensado para permitir comportamento dinâmico — responder a cliques, validar formulários, atualizar partes da página sem recarregar, animar elementos e muito mais. A ideia era dar aos desenvolvedores uma linguagem leve que rodasse diretamente no navegador do usuário.

Diferença entre execução no navegador e no servidor: Historicamente JavaScript rodava apenas no navegador (client-side), onde interage com a página e a interface do usuário. Com o surgimento do Node.js, JavaScript também passou a rodar no servidor (server-side). No navegador, o código tem acesso a APIs específicas como document (DOM) e ao modelo de eventos do navegador; no servidor, o ambiente fornece APIs de sistema de arquivos, rede, e processos—mas não há DOM nem document.

Conceito de sandbox no navegador: O sandbox é um ambiente restrito onde o código do navegador é executado para proteger o usuário. Isso significa que scripts em páginas web não podem, por exemplo, ler livremente arquivos do disco do usuário ou acessar recursos sensíveis fora das permissões definidas. O navegador isola (sandbox) cada origem (domínio) e aplica regras de segurança como CORS, políticas de mesma origem e permissões para APIs sensíveis (microfone, câmera, geolocalização).

Objetos globais comuns: no ambiente do navegador existem objetos e funções disponíveis globalmente, por exemplo:

  • document — representa o DOM (a página) e permite consultar e manipular elementos HTML.
  • console — usado para registrar mensagens de depuração no console do desenvolvedor (ex.: console.log()).
  • alert() — exibe uma caixa de diálogo simples com uma mensagem para o usuário.

Exemplos práticos

Exemplo: registrar uma mensagem no console usando console.log():

console.log('Olá do console!');

Exemplo: exibir uma saudação com alert(). Esta chamada será executada quando a página for carregada.

alert('Olá! Bem-vindo(a) à página sobre JavaScript.');

Exemplo: exibir a hora atual em um parágrafo usando document.getElementById().innerHTML:

document.getElementById('p-hora').innerHTML = 'Hora atual: ' + new Date().toLocaleString();

Hora atual: (a página irá preencher este campo ao carregar)

Nota: os exemplos acima são intencionais e simples para aprendizado. Em projetos reais prefira práticas como evitar alert() para UX, usar ferramentas de logging estruturado, e considerar frameworks e bundlers quando a aplicação crescer.
