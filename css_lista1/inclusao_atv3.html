Exemplo de CSS Interno

O CSS externo é colocado em um arquivo separado, como style.css, e o HTML se conecta a ele usando a tag <link>. É o método mais usado em sites grandes, pois organiza o código e separa estrutura de estilo.

Já o CSS inline é aplicado diretamente na tag HTML, usando o atributo style="". Exemplo: <p style="color: red;">. É prático para pequenos ajustes, mas não indicado para projetos grandes.
